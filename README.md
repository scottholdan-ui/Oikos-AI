# Oikos AI

Oikos AI is an AI-powered application builder designed around a team of specialized agents instead of a single AI.

## Vision

Users describe an idea and Oikos AI coordinates multiple expert agents to plan, design, build, test, and deploy advanced applications.

## Core Agents

- CEO Agent – product strategy and planning
- Architect Agent – system design
- Frontend Agent – UI implementation
- Backend Agent – APIs and infrastructure
- QA Agent – testing and validation
- DevOps Agent – deployment and monitoring

## MVP Goals

1. Idea intake chat
2. CEO planning workflow
3. Multi-agent task orchestration
4. Live application preview
5. File generation and code editing

## Long-Term Goal

Create an AI app builder that can compete with modern platforms while providing deeper planning, architecture, and development workflows.
